# Cadastro de Clientes – Console App (C#)

Projeto de **estudo em C#**, desenvolvido no meu **primeiro contato com a linguagem**, no início da minha carreira profissional, durante minha primeira experiência em empresa.

O objetivo é praticar **conceitos básicos de C#**, organização de código e boas práticas iniciais, utilizando uma aplicação de console.

---

## 🛠️ Tecnologias
- C#
- .NET (Console Application)
- Visual Studio

---

## ✅ Funcionalidades
- Cadastro de clientes
- Listagem de clientes
- Validação básica de dados de entrada

---

## 🎯 Objetivo
Projeto criado exclusivamente para **treinamento e aprendizado**, servindo como base para futuros estudos com banco de dados e frameworks.

- Remoção de clientes
